# Uso de Flyway

| flyway.conf        | Variable de entorno |
| ------------------ | ------------------- |
| `flyway.url`       | `FLYWAY_URL`        |
| `flyway.user`      | `FLYWAY_USER`       |
| `flyway.password`  | `FLYWAY_PASSWORD`   |
| `flyway.locations` | `FLYWAY_LOCATIONS`  |
| `flyway.schemas`   | `FLYWAY_SCHEMAS`    |
